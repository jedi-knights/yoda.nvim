HELP SYSTEM USAGE                                     *yoda-index-help-usage*

How to Use Vim Help:~

Navigation:~
    |topic|                       Follow link to topic
    <C-]>                         Follow link under cursor
    <C-o>                         Go back to previous location
    <C-i>                         Go forward to next location

Search:~
    /pattern                      Search in help file
    :helpgrep pattern             Search across all help files
    :cnext                        Next search result
    :cprev                        Previous search result

Quick Access:~
    K                             Help for word under cursor
    :help                         Open main help
    :help topic                   Help for specific topic

Tags:~
Help topics are surrounded by |tags| which you can jump to directly:
>
    :help yoda-keymaps            Open keymaps help
    :help yoda-python             Open Python guide
<
