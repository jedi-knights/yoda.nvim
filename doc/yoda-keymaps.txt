GENERAL NAVIGATION                                   *yoda-keymaps-navigation*

Legend: <leader> = <Space> (spacebar)
Modes: n = Normal, v = Visual, t = Terminal

Basic Movement ~
    <C-d>           (n,v)  Scroll down (cursor centered)
    <C-u>           (n,v)  Scroll up (cursor centered)
    n               (n)    Next search result (cursor centered)
    N               (n)    Previous search result (cursor centered)
    K               (n)    Hover docs (LSP) or Vim help for word under cursor

Window Navigation ~
    <C-h>           (n)    Move to left window
    <C-j>           (n)    Move to lower window
    <C-k>           (n)    Move to upper window
    <C-l>           (n)    Move to right window

FILE EXPLORER                                         *yoda-keymaps-explorer*

Snacks Explorer ~
    <leader>eo      (n)    Open explorer (only if closed)
    <leader>ef      (n)    Focus explorer (or open if closed)
    <leader>ec      (n)    Close explorer
    <leader>er      (n)    Refresh explorer
    <leader>ed      (n)    Debug explorer state
    <leader>e?      (n)    Show explorer keybindings help

In the Explorer ~
    H                      Toggle hidden files
    I                      Toggle ignored files
    h                      Close/collapse directory
    l                      Open directory or file

==============================================================================
SEARCH & FIND (fzf-lua)                               *yoda-keymaps-search*

    <leader>ff      (n)    Find files in current directory
    <leader>fg      (n)    Live grep (search text across files)
    <leader>fr      (n)    Recent files
    <leader>fb      (n)    Find open buffers
    <leader>fR      (n)    Document symbols (LSP, current file)
    <leader>fS      (n)    Workspace symbols (LSP, all files)
    <leader>fD      (n)    Browse all diagnostics
    <leader>fG      (n)    Find git-tracked files

==============================================================================
LSP & CODE INTELLIGENCE                                   *yoda-keymaps-lsp*

Navigation ~
    gd              (n)    Go to definition
    <leader>ld      (n)    Go to definition
    <leader>lD      (n)    Go to declaration
    <leader>li      (n)    Go to implementation
    <leader>lr      (n)    Find references

Code Actions ~
    <leader>la      (n)    Code actions
    <leader>lrn     (n)    Rename symbol
    <leader>lf      (n)    Format buffer

Symbols ~
    <leader>ls      (n)    Document symbols (current file)
    <leader>lw      (n)    Workspace symbols

Diagnostics ~
    <leader>le      (n)    Show diagnostic float (cursor)
    <leader>lq      (n)    Add diagnostics to location list
    [d              (n)    Previous diagnostic
    ]d              (n)    Next diagnostic

AI & CODE ASSISTANCE                                       *yoda-keymaps-ai*

Claude Code Integration ~
    <leader>ai      (n)    Toggle Claude Code terminal
    <leader>af      (n)    Focus Claude Code window
    <leader>ar      (n)    Resume previous Claude session (--resume)
    <leader>aC      (n)    Continue last Claude conversation (--continue)
    <leader>am      (n)    Select Claude model

Context ~
    <leader>aB      (n)    Add current buffer to Claude context
    <leader>as      (v)    Send visual selection to Claude
    <leader>as      (n)    Add file to Claude (in file explorer buffers)

Diffs ~
    <leader>aa      (n)    Accept diff from Claude
    <leader>ad      (n)    Deny diff from Claude

See |yoda-ai| for setup instructions.

TESTING                                              *yoda-keymaps-testing*

Neotest ~
    <leader>ta      (n)    Run all tests (cwd)
    <leader>tn      (n)    Run nearest test
    <leader>tf      (n)    Run current file tests
    <leader>tl      (n)    Run last test
    <leader>tc      (n)    Run current file (filetype-aware)
    <leader>td      (n)    Debug nearest test (DAP)
    <leader>ts      (n)    Toggle test summary panel
    <leader>to      (n)    Toggle test output panel
    <leader>tv      (n)    View test output (enter)
    <leader>tO      (n)    Focus test output panel
    <leader>tF      (n)    Focus test summary panel
    <leader>tC      (n)    Clear test output panel

pytest-atlas ~
    <leader>tS      (n)    Show test environment status
    <leader>tL      (n)    Open pytest-atlas log (tail, last 100 lines)
    <leader>tLA     (n)    Open pytest-atlas log (full)
    <leader>tX      (n)    Clear pytest-atlas log

==============================================================================
GIT INTEGRATION                                           *yoda-keymaps-git*

    <leader>gg      (n)    Open Neogit (interactive git interface)
    <leader>gB      (n)    Git blame full file (Fugitive)

Gitsigns hunk operations are registered as buffer-local keymaps inside
the gitsigns on_attach callback. Use :map <buffer> to inspect them.

WINDOW MANAGEMENT                                     *yoda-keymaps-windows*

Splits ~
    <leader>|       (n)    Vertical split
    <leader>-       (n)    Horizontal split
    <leader>ws      (n)    Equalize window sizes

Diagnostics / Trouble ~
    <leader>xt      (n)    Focus Trouble diagnostics window
    <leader>xT      (n)    Show all TODO comments in Trouble

==============================================================================
UTILITIES & NOTIFICATIONS                           *yoda-keymaps-utilities*

General ~
    <leader>qq      (n)    Quit Neovim
    <leader>D       (n)    Delete buffer content (void register, no clipboard)
    <leader>tu      (n)    Toggle conform.nvim auto-format on save
    <leader>d       (n)    Open Snacks dashboard
    <leader><leader>r (n)  Hot reload Yoda config

Keymap Discovery ~
    <leader>kk      (n)    Show all leader keymaps (float window)
    <leader>sk      (n)    Toggle Showkeys screencaster display

Noice / Notifications ~
    <leader>nm      (n)    Show message history (Noice)
    <leader>nl      (n)    Show last message
    <leader>nh      (n)    Show notification history
    <leader>nd      (n)    Dismiss all notifications

==============================================================================
BUFFER NAVIGATION                                     *yoda-keymaps-buffers*

    ]b              (n)    Next buffer
    [b              (n)    Previous buffer

==============================================================================
VISUAL MODE                                             *yoda-keymaps-visual*

    <leader>r       (v)    Start substitute command on selection
    <leader>y       (v)    Yank selection to system clipboard
    <leader>d       (v)    Delete to void register (no clipboard change)
    <leader>p       (v)    Replace selection, preserve clipboard

==============================================================================
DISABLED KEYS                                          *yoda-keymaps-disabled*

The following keys are disabled to encourage better vim habits:

    <up>            (n)    Use 'k' instead
    <down>          (n)    Use 'j' instead
    <left>          (n)    Use 'h' instead
    <right>         (n)    Use 'l' instead
    <pageup>        (n)    Use '<C-u>' instead
    <pagedown>      (n)    Use '<C-d>' instead

Note: jk → <Esc> insert-mode mapping was removed — it caused a 300ms stall
on every 'j' typed. Use <C-[> or physical Escape instead.
